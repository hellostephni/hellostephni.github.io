CSS Display and Positioning Reference

CSS Properties

Property Description
position Determines position of HTML elements on a web page.
z-index Specifies how far back or how far forward an element will appear on a web page when it overlaps other elements.
display Determines the type of render block for an element.
float Determines how far left or how far right an element should float within its parent element. To use this property, width of container must be specified.
clear Specifies how an element should behave when it bumps into another element within the same containing element. Usually used in combination with elements having the CSS float property.

CSS Values

Declaration Description
Property Value
position fixed The element is set/pinned to a specific spot on a page, stays in the same space regardless of scrolling.
relative Enables an element to be positioned relative to where it would have originally been on a web page. Property then has to be further edited with offset propertiies to determine the actual position. Without the offset properties, this declaration will have no effect on its positioning.
absolute Removes an element entirely from the document flow and positions it relative to its closest parent element that is positioned with relative or absolute. Element can then be further motified with positioning attributes top, left, bottom and right.
static Default value.
z-index n Uses positive or negative integer values. Element with the highest z-index value will be at the foreground, while the element with the lowest z-index value will be at the back.
display block Elements take up the full width of their container with line breaks before and after, and can have their height and width manually adjusted.
inline Elements take up as little space as possible, flow horizontally, and cannot have their width or height manually adjusted.
inline-block Elements can appear next to each other, and can have their width and height manually adjusted.
float left Floats an element to the left side of its container.
right Floats an element to the right side of its container.
clear left Determines that no other elements within the same containing element are allowed to float on the left side of this element.
right Determines that no other elements within the same containing element are allowed to float on the right side of this element.
both Determines that no elements within the same containing element are allowed to float on either side of this element.
